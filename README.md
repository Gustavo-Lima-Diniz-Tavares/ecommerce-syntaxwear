# SyntaxWear - E-commerce de Tênis e Sneakers

Bem-vindo ao repositório da **SyntaxWear**, uma plataforma moderna e elegante voltada para a venda online de tênis e sneakers exclusivos. O projeto foi desenvolvido com foco em performance, design responsivo e uma experiência de usuário fluida.

---

## 🚀 Sobre o Projeto

A SyntaxWear é um site de e-commerce fictício desenvolvido para demonstrar habilidades em estruturação de código front-end, organização de componentes CSS e design responsivo. O site apresenta coleções divididas por categorias (Casual, Esporte, Moderno, Futurista) e destaca o modelo "Krypton One".

### ✨ Funcionalidades Principais

- **Design Responsivo:** Adaptado para dispositivos móveis, tablets e desktops.
- **Menu Mobile:** Menu de navegação estilo hambúrguer para dispositivos menores.
- **Categorias de Produtos:** Navegação facilitada por estilos de vida e uso.
- **Grade de Produtos (Grid):** Layout dinâmico para exibição de destaques e novos modelos.
- **Seção Hero:** Banner principal impactante com chamadas para ação (CTAs).
- **Newsletter:** Área dedicada para captura de e-mails no rodapé.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno:

- **HTML5:** Estruturação semântica de todo o conteúdo.
- **CSS3:** Estilização avançada com o uso de:
  - **CSS Variables:** Para fácil manutenção de cores, fontes e espaçamentos.
  - **Flexbox & CSS Grid:** Para criação de layouts complexos e responsivos.
  - **Componentização:** CSS organizado em arquivos separados por componentes para melhor escalabilidade.
- **Imagens Otimizadas:** Uso de SVGs para ícones e logotipos, garantindo nitidez em qualquer resolução.

---

## 📁 Estrutura de Arquivos

A organização do projeto segue uma estrutura modular para facilitar a manutenção:

```text
ecommerce-syntaxwear/
├── css/
│   ├── components/       # Estilos específicos de cada seção (header, footer, hero, etc.)
│   ├── base.css          # Estilos globais e botões
│   ├── reset.css         # Reset de estilos padrão do navegador
│   ├── variables.css     # Definição de cores e fontes (Custom Properties)
│   └── layout.css        # Estrutura principal de containers
├── images/
│   ├── banners/          # Imagens de fundo e banners principais
│   ├── icons/            # Ícones do site em formato SVG
│   ├── logo/             # Logotipo oficial
│   └── products/         # Fotos dos produtos e categorias
├── index.html            # Arquivo principal do site
└── README.md             # Documentação do projeto
```

## 💻 Como Visualizar o Projeto

Como este é um projeto estático de front-end, você não precisa instalar dependências complexas.

1. Clone este repositório:
   ```bash
   git clone https://github.com/Gustavo-Lima-Diniz-Tavares/ecommerce-syntaxwear
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd ecommerce-syntaxwear
   ```
3. Abra o arquivo `index.html` em seu navegador de preferência ou utilize a extensão **Live Server** no VS Code para uma melhor experiência.
